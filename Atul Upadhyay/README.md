5. Word Ladder Problem

A word ladder is a sequence of words where each word differs from the previous word by exactly one letter.
You are given two words, beginWord and endWord, and a dictionary of words.
Return the length of the shortest transformation sequence from beginWord to endWord.
If no such transformation sequence exists, return 0.

Example 1

Input:
beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log","cog"]

Output:
5

Explanation:
One shortest transformation is "hit" → "hot" → "dot" → "dog" → "cog".

Example 2

Input:
beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log"]

Output:
0
