Question 3 : What is feature engineering, and why is it important for machine learning?

Answer: 

Feature engineering is the process of transforming raw data into meaningful features (inputs) that help a machine learning model learn better and make more accurate predictions.

In simple words — it’s about making your data smarter before feeding it to the model.

For example:

- If you have a date column, you can extract useful features like day of the week, month, or year. 

- If you have a person’s age, you might group it into age ranges (like “teen,” “adult,” “senior”).

- You can also create new features by combining existing ones, like price per square foot from price and area.

Why it’s important:

- Good features help the model understand the data’s patterns more easily.

- It improves model performance and accuracy.

- It reduces the need for overly complex models — sometimes, better features can outperform fancy algorithms.
