Question 1 : Explain the concept of a training set, validation set, and test set.

Answer: 

Training Set: This is the data the model learns from. It’s like the lessons in a classroom — the model studies this to understand patterns.

Validation Set: This is the data we use to check and fine-tune the model while it’s learning. Think of it as a practice test — it helps you see if the model is doing well or if it needs some adjustments.

Test Set: This is the data we use at the very end to see how the model performs on completely new stuff. It’s like the final exam — it tells us how the model will do in the real world.
